# 數位孿生應用
此專案介紹數位孿生技術在工業系統中的應用，著重於物理系統與虛擬模型的同步，並結合 IoT 和機械手臂實現全方位的系統監控與操作。

---

## 功能與角色

數位孿生的核心功能包括：
1. **實時監控**：
   - 將物理系統的數據同步到虛擬模型，實現即時狀態更新。
2. **模擬與分析**：
   - 基於虛擬環境，進行設備的行為模擬和性能預測。
3. **故障檢測與優化**：
   - 結合缺陷檢測模組，快速識別異常情況並提供優化建議。
4. **交互操作**：
   - 與機械手臂互動，完成分類、搬運等操作，並同步結果至虛擬環境。
